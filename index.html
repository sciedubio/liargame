🚀 나만의 단원 연동하는 법

구글 시트 하나만 작성해두면 학생들은 버튼만 눌러서 모든 단원 게임을 할 수 있습니다!

  1. 구글 시트 생성 후 A열:단원명, B열:제시어, C열:설명, D열:가짜단어, E열(선택):시크릿단어 입력
  2. 메뉴에서 [파일]-[공유]-[웹에 게시] 클릭
  3. '전체 문서'를 'CSV'로 변경 후 게시
  4. 복사된 주소를 HTML 소스코드의 MASTER_CSV_URL에 넣고 배포하세요!

📥 결과 점수 자동 수집 (선택)

결과를 모을 새 시트를 열고 [확장 프로그램]-[Apps Script]에 아래 코드를 넣은 뒤 '웹 앱'으로 배포하세요. 그 URL을 SCORE_SAVE_URL에 넣으면 끝!

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  if (sheet.getLastRow() === 0) sheet.appendRow(['Date', 'Group', 'Theme', 'Rounds', 'Player Data']);
  try {
    const data = JSON.parse(e.postData.contents);
    sheet.appendRow([data.date, data.group, data.theme, data.rounds, JSON.stringify(data.players)]);
    return ContentService.createTextOutput(JSON.stringify({"status": "success"})).setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({"status": "error"})).setMimeType(ContentService.MimeType.JSON);
  }
}
